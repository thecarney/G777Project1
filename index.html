Description
Users can specify a Power value and the tool will use an Inverse Distance Weighted model to create a prediction surface for nitrate concentration within each bin in the hex map. A continuous linear regression is then performed to explore whether nitrate concentration is a good explanatory variable for cancer rates given the provided Power value.

The layers widget provides for exploratory interaction with the results. The Cancer Rate layer is included as an overlay which may be revealed using the slider handle.
